A manifest for a mod. The same mod meant for different mod loaders (e.g. Quilt, Fabric, Forge, etc.) will have different manifests.

Author

ReviversMC

Since

1.0.0-2.0.0

Parameters

schemaVersion

The version of the manifest schema.

fancyName

The user readable name of the project.

author

The author/publisher of the mod.

license

The license of the mod, or a url if custom.

curseForgeId

The curseforge id of the mod, not the slug.

modrinthId

The modrinth id of the mod, not the slug.

links

A list of links related to the mod.

files

File versions for the mod.
